# Josbet Florist - Proyecto

Estructura básica creada. Reemplaza las imágenes en `images/` con las que quieras usar.

Nombres de archivo esperados para el fondo y galería (colócalas en `josbet-florist/images/`):
- `bg_welcome.jpg` (imagen para la página de bienvenida)
- `bg_views.jpg` (imagen de fondo para las demás vistas)
- `sunflower.jpg` (girasol)
- `tulip.jpg` (tulipanes)
- `rose.jpg` (rosas)
- `single1.jpg`, `single2.jpg`, `single3.jpg` (flores individuales)

Abrir `index.html` en el navegador para comenzar.

Ejecutar el script para crear/verificar carpetas (PowerShell):

```powershell
# desde PowerShell en Windows
cd D:\FLORERIA-BARTOLO\josbet-florist
.\setup-create-folders.ps1
```

Si prefieres crear manualmente, usa los nombres de carpeta indicados (`css`, `js`, `images`, `assets`).
